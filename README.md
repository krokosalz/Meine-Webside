# Bewerbungs-Website von Nicola Schmied

Live: https://krokosalz.github.io/Meine-Webside/

## Zweck
Persönliche Bewerbungswebsite mit CV, Motivationsschreiben, Projekten und Kontakt.

## Seiten
- Start: Überblick + Links
- CV: Erfahrung, Skills, Ausbildung
- Motivation: Motivationsschreiben
- Projekte: Projektübersicht + GitHub Links
- Kontakt: Kontaktmöglichkeiten + Social Links

## Tech Stack
- HTML5
- CSS3

## Projektstruktur
- `index.html` – Startseite
- `pages/` – Unterseiten (cv, motivation, projects, contact)
- `css/` – Stylesheet(s)
- `img/` – Bilder
- `assets/` – Downloads (z.B. CV als PDF)

## Lokal starten
Option 1: Datei direkt öffnen  
- `index.html` im Browser öffnen

Option 2: VS Code Live Server  
- Rechtsklick auf `index.html` → **Open with Live Server**

## Deployment
Gehostet mit GitHub Pages (Branch: `main`, Folder: `/ (root)`).
